About

Graph Neural Networks (GNN) are a class of deep learning methods designed to perform inference on data described by graphs. GNNs can be directly applied to graphs, and provide an easy way to do node-level, edge-level, and graph-level prediction tasks. This API allows the user to build a specific variant of GNNs called graph convolutional network (GCNs) [1] to solve node classification tasks by simply formatting and uploading their data. Result metrics and a summary of the data is provided at the end. Currently, the intention is to get a feel for the potential performance of the model rather than getting the most optimized version. However, that is the one of the future directions of the project.

It takes a few minutes to build a model for a dataset 3000 nodes and 6000 edges. The API can currently handle datasets with a size of 5-10 MB. Keep in mind that it has not been tested for very large datasets and the resources may be not be sufficient for such analysis.

[1] Kipf & Welling, Semi-Supervised Classification with Graph Convolutional Networks, 2016
